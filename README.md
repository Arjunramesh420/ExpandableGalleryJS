# Expansible Image Gallery Component

A React component for creating an expansible image gallery with toggleable panels. Clicking on a panel expands it while collapsing others.

## Usage

1. Install dependencies:
   ```bash
   npm install
Run the application:

bash
Copy code
npm start
Access the gallery on http://localhost:3000

Features
Toggleable Panels: Clicking on a panel expands it and collapses others.
Customization: Easily customize the content and background images of each panel.
